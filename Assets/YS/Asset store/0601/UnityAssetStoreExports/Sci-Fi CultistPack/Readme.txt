This asset pack contains 3 main sprite groups, 3 groups of Cultists divided by pallet type/color grouping, 7 groups of "Statues" divided by color/pallet
and 16 sprites of Bots/Minions with different styles and colors.
All assets are contained in one single sprite sheet, if you wish not to use any number of them you may simply delete the selected spites in the sprite editor
